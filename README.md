# 📄 Resume Builder with Redux Thunk 

## 📌 Overview
A modern Resume Builder application that allows users to create, customize, and preview professional resumes in real-time. Built with scalable architecture using Redux Toolkit and Thunks for efficient state management and dynamic data handling.

---

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Redux Thunk
- Tailwind CSS
- JavaScript (ES6+)

---

## ✨ Features
- 📝 Dynamic resume form with multiple sections (Education, Experience, Skills)
- 👀 Real-time resume preview while editing
- 📄 Structured resume layout generation
- 🔄 State management using Redux Toolkit & Thunks
- ⚡ Async handling for dynamic data updates
- 🧩 Modular and reusable component architecture
- 📱 Fully responsive UI

---

## 🧠 Key Learnings
- Managing complex multi-section forms in React
- Implementing async logic using Redux Thunks
- Real-time UI synchronization between form and preview
- Structuring scalable frontend architecture
- Handling dynamic data flow in large applications
