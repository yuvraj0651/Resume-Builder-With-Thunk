# 📄 Resume Builder App

## 📌 Overview
An interactive resume builder that allows users to create, preview, and download professional resumes in real-time.

## 🛠️ Tech Stack
- React.js
- Tailwind CSS

## ✨ Features
- ✍️ Dynamic form inputs
- 👀 Live preview
- 📄 Resume layout generation
- 📱 Responsive UI

## 🧠 Key Learnings
- Form handling
- Real-time UI updates
- Component reusability

## 🚀 Future Improvements
- PDF download
- Multiple templates
- User authentication
